# MP42PDF 视频转分镜小工具

![让AI替我学画画吧](README.files/让AI替我学画画吧.jpg)

丢进去一个视频，吐出来一个分镜PDF。就这样。

![pdf_example](README.files/pdf_example.png)

抓人物动态、学构图、分析剧情结构，或者五分钟速通一整季番剧。
既然AI能替我们打工，不如也让它替我们看番。

---

## 下载

👉 [点这里下载 mp42pdf.exe](../../releases)

（Windows，不用装任何东西，双击就能用）

---

## 怎么用

把视频文件拖到 `mp42pdf.exe` 上面，完事。

或者双击打开，点 Browse 选视频，再点生成按钮。

PDF 会自动保存在视频旁边。
